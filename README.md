# Conversor de Unidades
Clona el repositorio y ejecuta el programa:

git clone https://github.com/denislcian/mi-proyecto-evolve-1.git

cd mi_proyecto/src

python main.py

## Funcionalidades

-Metros a kilómetros

-Pulgadas a centímetros

-Celsius a Fahrenheit

-Gramos a kilos
